Software

Our lab publishes open source software related to our research projects. We currently have published:

  • WiFu

    A composable toolkit for experimental wireless transport protocols, from the WiFu project.

  • omnet-extension and inet-extension

    Code for OMNET++ that enables the simulator to interact with the decision engine of a streaming video player, from the Dynamic Adaptive Streaming over HTTP project.

We also run the BYU Open Source Lab, where we create free software for family history and civic apps.
